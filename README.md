# WinDOS
A Project to make an "OS" in Batch/CMD. Still developing. 
I have coded a bunch of "command" in this "OS"

Command to change theme:
Note: You can only change theme at the Settings:

$theme-black
$theme-blue 
$theme-aqua
$theme-red
$theme-green
$theme-purple
$theme-yellow
$theme-white
$theme-white

Commands to use at Command Window:

Power Commands:
power/shutdown
power/hibernate
power/restart 

Net Commands:
net/spdplan
net/wifiimpexp
